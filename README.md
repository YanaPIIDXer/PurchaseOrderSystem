# PurchaseOrderSystem
商品発注書作成システム。React + ASP.NETのポートフォリオ。

## このシステムで実現する事

- 管理者
  - 業者の管理
  - 発注書の作成
  - 作成した発注書を印刷用にダウンロード
- 業者
  - 商品の管理
